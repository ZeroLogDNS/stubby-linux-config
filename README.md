## Dependencies:

**curl**

**stubby** (automatic install and config)

**NetworkManager**

## Init Support:

**SystemD**

**Runit**

**Dinit**

**Openrc**

**s6**

**suite66**

## OS Support:

**Artix**

**Arch**

**Arco Linux**

**CentOS**

**Fedora**

**SolusOS**

**Void**

**Debian**

**Ubuntu**

*Nearly all Debian based system with NetworkManager*

*Nearly all Arch based system with NetworkManager*

## Unsupported:

**Raspberry PI OS**

**MX Linux (Sysvinit)**
